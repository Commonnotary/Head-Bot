# Client Onboarding Checklist for Head-Bot

**Purpose:** Ensure smooth transition for newly signed clients and establish strong foundation for long-term partnership.

**Responsible Party:** Dedicated Account Manager
**Timeline:** 30-day onboarding window

---

## PHASE 1: PRE-ONBOARDING (Before Contract Signed)

### Day 0-1 (Contract Preparation)

- [ ] **Finalize Service Agreement**
  - [ ] Document pricing and volume assumptions
  - [ ] Confirm service levels (standard/expedited/rush)
  - [ ] Document turnaround timelines
  - [ ] Specify payment terms and due date
  - [ ] Include liability and guarantee terms
  - [ ] Obtain all required signatures

- [ ] **Prepare Welcome Email**
  - [ ] Customize with company/contact name
  - [ ] Get user approval before sending
  - [ ] Include onboarding timeline overview
  - [ ] Outline next steps

- [ ] **Assign Account Manager**
  - [ ] Determine account manager for client
  - [ ] Create CRM account record for client
  - [ ] Set up contact list (primary, secondary, AP)
  - [ ] Configure billing and payment method
  - [ ] Flag any special requirements or notes

---

## PHASE 2: WEEK 1 ONBOARDING (After Contract Signed, Before First Order)

### Task 1: Immediate Communication (Day 1)

- [ ] **Send Welcome Email**
  - [ ] Confirm receipt of signed contract
  - [ ] Introduce dedicated account manager
  - [ ] Overview of onboarding timeline
  - [ ] Confirm preferred contact method and response times
  - [ ] Next step: Portal setup meeting

**Email from:** Account Manager  
**Approval Required:** Yes (before sending)

---

### Task 2: Portal Setup & Training (Day 1-2)

- [ ] **Create Client Portal Account**
  - [ ] Generate unique login credentials
  - [ ] Set up secure password (temporary password)
  - [ ] Configure user roles (admin, submitter, approver)
  - [ ] Add primary and secondary contacts to account
  - [ ] Test portal access from client IP

- [ ] **Prepare Portal Training Materials**
  - [ ] Create step-by-step portal user guide (PDF)
    - Screenshots showing how to:
      * Log in
      * Upload documents
      * Check status
      * Download completed apostilles
      * Submit support requests
  
  - [ ] Create video walkthrough (10-15 min) OR schedule live demo
  - [ ] Document keyboard shortcuts and tips
  - [ ] Include troubleshooting FAQ

- [ ] **Schedule Portal Training**
  - [ ] Propose 2-3 time slots for live training
  - [ ] Include all relevant team members (admin, staff uploaders)
  - [ ] Send calendar invite with Zoom/Teams link
  - [ ] Estimate 30 minutes for walkthrough + Q&A

---

### Task 3: Document Submission Guide (Day 2)

- [ ] **Create Custom Document Checklist**
  - [ ] List common documents they'll submit (based on industry)
    - Example: "Corporate Resolutions," "Diploma Certificates," etc.
  
  - [ ] For each document type, include:
    - [ ] Definition and use case
    - [ ] Quality requirements (legible, complete pages, etc.)
    - [ ] Common mistakes to avoid
    - [ ] Example image or template

  - [ ] Create apostille requirements by state/country they operate in
  - [ ] Include processing timelines per document type
  - [ ] Provide sample completed apostille document

- [ ] **Create Submission Guidelines**
  - [ ] File format requirements (PDF, JPG, etc.)
  - [ ] File size limits
  - [ ] Batch submission recommendations
  - [ ] Quality standards (resolution, color, legibility)
  - [ ] Special handling instructions (original vs. copies)

**Approval:** Account manager review for accuracy; user approval before sending to client

---

### Task 4: Initial Account Configuration (Day 2-3)

- [ ] **Set Up CRM Fields**
  - [ ] Primary contact name and email
  - [ ] Secondary contact name and email
  - [ ] AP/Billing contact name and email
  - [ ] Preferred communication channel (email/phone)
  - [ ] Best time to reach (morning/afternoon)
  - [ ] Estimated monthly volume
  - [ ] Contract start/end date
  - [ ] Annual contract value (ACV)
  - [ ] Service level (standard/expedited/premium)
  - [ ] Account manager name

- [ ] **Set Up Billing Configuration**
  - [ ] Payment method on file (credit card/ACH/invoice)
  - [ ] Billing email address
  - [ ] Billing cycle (monthly/quarterly/annual)
  - [ ] PO number requirement (if applicable)
  - [ ] Tax exemption (if applicable)

- [ ] **Create Communication Preferences**
  - [ ] Email notification settings
  - [ ] Invoice delivery method
  - [ ] Preferred contact method for urgencies
  - [ ] Do-not-contact times
  - [ ] Escalation path

---

## PHASE 3: WEEK 2 ONBOARDING (Portal Live, First Orders Preparation)

### Task 5: Pilot Order (Day 8-10)

- [ ] **Prepare Pilot Order Instructions**
  - [ ] Identify 3-5 test documents for first batch
  - [ ] Send list to client with:
    - [ ] Document type
    - [ ] Expected turnaround (standard timeline)
    - [ ] Submission deadline for pilot
    - [ ] Quality expectations

- [ ] **Set Up Pilot Order Tracking**
  - [ ] Create task: "Monitor pilot documents" (high priority)
  - [ ] Schedule daily status checks
  - [ ] Alert account manager to any issues immediately
  - [ ] Plan quality review on day 1 of processing

- [ ] **Execute Pilot Order Flawlessly**
  - [ ] Receive pilot documents
  - [ ] Verify completeness and quality
  - [ ] Process with highest attention to detail
  - [ ] Double-check all apostilles before delivery
  - [ ] Deliver with clear status communication

- [ ] **Gather Feedback**
  - [ ] Call or email client within 24 hours of delivery
  - [ ] Ask specific questions:
    - "Did the documents meet your needs?"
    - "Was the turnaround acceptable?"
    - "Any issues with the portal or process?"
    - "What went well? What could improve?"
  
  - [ ] Document feedback in CRM
  - [ ] Implement quick wins before full volume

---

### Task 6: First Billing Statement (Day 7-15)

- [ ] **Generate First Invoice**
  - [ ] Calculate charges based on pilot order
  - [ ] Itemize by document type
  - [ ] Include any setup fees or credits
  - [ ] Confirm payment terms
  - [ ] Include payment instructions

- [ ] **Send Invoice**
  - [ ] Send via preferred method (email/portal)
  - [ ] Include cover email with:
    - [ ] Thank you message
    - [ ] Invoice overview
    - [ ] Payment deadline
    - [ ] Contact for billing questions
    - [ ] Link to portal for viewing orders

- [ ] **Process Payment**
  - [ ] Confirm receipt within 2 business days
  - [ ] Log payment in accounting system
  - [ ] Send payment confirmation to client

**Payment Processing:** Account manager confirms; keep records in CRM

---

## PHASE 4: WEEK 3-4 ONBOARDING (Smooth Operations)

### Task 7: Ongoing Communication Setup (Day 15-22)

- [ ] **Establish Check-In Rhythm**
  - [ ] Schedule weekly check-in calls (first 4 weeks)
    - [ ] Goal: Quick sync on volume, questions, satisfaction
    - [ ] Duration: 15-20 minutes
    - [ ] Time: Client's preferred day/time
  
  - [ ] After 4 weeks, move to 2x per month
  - [ ] Send calendar invites for all calls
  - [ ] Create task reminders (1 day before each call)

- [ ] **Create Monthly Business Review Agenda**
  - [ ] Volume summary (documents processed)
  - [ ] Turnaround performance (on-time delivery %)
  - [ ] Cost summary (total spent YTD)
  - [ ] Satisfaction check ("How are we doing?")
  - [ ] Upcoming needs (plan ahead)
  - [ ] Special requests or improvements

- [ ] **Set Up Automated Reminders**
  - [ ] Monthly check-in task (1st of each month)
  - [ ] Quarterly business review (every 3 months)
  - [ ] Contract renewal reminder (30 days before expiration)
  - [ ] Annual satisfaction survey (yearly)

---

### Task 8: Training & Support Resources (Day 15-22)

- [ ] **Create Client Knowledge Base**
  - [ ] How-to: Submit documents via portal
  - [ ] How-to: Track order status
  - [ ] How-to: Download completed apostilles
  - [ ] How-to: Request additional copies
  - [ ] How-to: Get help or escalate issues
  - [ ] FAQ: Common questions and answers
  - [ ] Troubleshooting: Portal login issues, upload errors, etc.

- [ ] **Provide Reference Documents**
  - [ ] State-by-state apostille requirements guide
  - [ ] Industry-specific templates (if applicable)
  - [ ] List of documents NOT eligible for apostille
  - [ ] Sample completed apostille (marked-up with annotations)
  - [ ] Quality checklist (what we look for)

- [ ] **Assign Support Contact**
  - [ ] Name and phone of primary support person
  - [ ] Email for urgent issues
  - [ ] Support hours (Monday-Friday, [times])
  - [ ] Emergency escalation number
  - [ ] SLA for response times

**Resource Approval:** Account manager + user review before finalizing

---

### Task 9: Service Level Expectations (Day 20-25)

- [ ] **Confirm SLA (Service Level Agreement)**
  - [ ] Turnaround time per document type
  - [ ] Quality guarantee (error rate, rework policy)
  - [ ] Support response time (24 hours standard)
  - [ ] Escalation procedures for urgent issues
  - [ ] Refund/credit policy if service not met

- [ ] **Document in CRM**
  - [ ] Add SLA terms to client record
  - [ ] Create task to monitor compliance
  - [ ] Set up alerts if SLA at risk

- [ ] **Review with Client**
  - [ ] Send SLA document via email
  - [ ] Confirm understanding on next check-in call
  - [ ] Address any concerns or exceptions

---

## PHASE 5: 30-DAY ONBOARDING COMPLETION

### Task 10: Onboarding Completion Review (Day 28-30)

- [ ] **Conduct 30-Day Check-In Call**
  - [ ] Schedule 30-minute call with primary contact
  - [ ] Review onboarding process (what went well, what to improve)
  - [ ] Confirm system is working smoothly
  - [ ] Address any outstanding questions
  - [ ] Discuss expected volume for next month
  - [ ] Ask for testimonial or case study permission

**Script from:** PHONE_SCRIPTS.md "Month 1 Relationship Check-in" [custom for this client]

- [ ] **Document Feedback**
  - [ ] Log conversation notes in CRM
  - [ ] Flag any process improvements needed
  - [ ] Share feedback with operations team
  - [ ] Update account record with satisfaction score

- [ ] **Process Feedback Internally**
  - [ ] Review onboarding with team
  - [ ] Identify what worked well
  - [ ] Identify improvements for next client
  - [ ] Update templates/processes if needed

- [ ] **Transition to Standard Operations**
  - [ ] Move from weekly to 2x monthly check-ins
  - [ ] Reduce account manager touchpoints to normal
  - [ ] Confirm client is independent on portal
  - [ ] Set up recurring calendar events (monthly check-in)

---

### Task 11: Post-Onboarding Relationship Foundation (Day 30+)

- [ ] **Establish Long-Term Touch Points**
  - [ ] Monthly check-in email (non-transactional)
  - [ ] Quarterly business review call (30 minutes)
  - [ ] Annual satisfaction survey
  - [ ] Contract renewal review (60 days before expiration)
  - [ ] Annual rate increase notification (if applicable)

- [ ] **Create Account Expansion Opportunity List**
  - [ ] Document additional services they might need
  - [ ] Identify cross-sell opportunities:
    - [ ] Certified copies (if ordering apostilles)
    - [ ] Multi-state apostilles (if expanding geographically)
    - [ ] Bulk discounts (if volume growing)
    - [ ] Retainer agreements (if consistent monthly volume)
  
  - [ ] Schedule follow-up on expansion items (quarterly)

- [ ] **Secure Testimonial / Case Study**
  - [ ] If satisfied, request permission to use as reference
  - [ ] Record specific results:
    - [ ] Documents processed
    - [ ] Time saved
    - [ ] Cost comparison to previous vendor
    - [ ] Satisfaction rating
  
  - [ ] Create case study document (optional)
  - [ ] Get written approval before using publicly

## ONBOARDING SUCCESS METRICS

By end of 30-day period, client should achieve:

✅ **Adoption Metrics:**
- [ ] Portal login successful (primary + secondary users)
- [ ] First order submitted via portal
- [ ] Payment processed without issues
- [ ] Client confident using portal independently

✅ **Service Metrics:**
- [ ] All pilot orders delivered on-time
- [ ] Zero quality issues on first orders
- [ ] 100% accuracy rate on apostilles
- [ ] Response time <24 hours on all inquiries

✅ **Relationship Metrics:**
- [ ] Weekly check-ins attended (4 calls = 4 attended)
- [ ] Client feedback gathered and documented
- [ ] Client satisfaction score ≥4/5
- [ ] No escalated issues or complaints

✅ **Pipeline Metrics:**
- [ ] Next month forecast documented
- [ ] Expansion opportunities identified
- [ ] Renewal date documented in CRM
- [ ] Referral permission obtained

## COMMON ONBOARDING ISSUES & SOLUTIONS

| Issue | Root Cause | Solution |
|-------|-----------|----------|
| **Slow portal adoption** | Unclear instructions, time needed to learn | Offer live training, create video tutorial, follow up weekly |
| **First order quality issues** | Incomplete/poor quality submission | Provide detailed submission checklist, review pilot strictly |
| **Billing confusion** | Unexpected charges, unclear invoicing | Itemize invoice, confirm terms before processing, send invoice early |
| **High support tickets early** | Unclear expectations or process issues | Increase check-in frequency, provide comprehensive FAQ |
| **Delayed first payment** | Budget approval, payment method issues | Confirm payment method before contract, send invoice within 5 days |
| **Low engagement** | Client uncertain about need, distracted | More frequent check-ins, find internal champion, share ROI data |

To mark onboarding complete:

1. **Account Manager** completes all checklist items above

3. **Account Manager** schedules review with operations team
4. **Operations Manager** approves completion
5. **Account Manager** sends "graduation email" to client
6. **Move to standard account management** (monthly cadence)
